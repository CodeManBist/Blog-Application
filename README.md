# VTU Internship Project

Full-stack web application built using Django (Backend) and React (Frontend).

## Tech Stack
- Backend: Django, Django REST Framework
- Frontend: React (Vite)
- Database: SQLite (development)
- Version Control: Git & GitHub

## Project Structure
/backend → Django REST API  
/frontend → React UI  

## Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm run dev
